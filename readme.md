## 프로젝트 개요

플러그를 원격으로 제어해보는 것은 어떨까? 라는 생각으로 만들어진 IOT 플러그 프로젝트입니다.

## 프로젝트 상세설명

사용자가 회원가입하고, 플러그의 고유 아이디 값을 이용하여 기기 등록을 하면, 플러그를 제어할 수 있습니다.
